# Biomechanical Gait Analysis in Python

## Overview

This repository contains a Python-based reimplementation of a biomechanics gait analysis assignment.

The original laboratory work was reworked independently to strengthen scientific programming skills while applying engineering principles to human movement analysis.

The project includes data processing, numerical calculations, graphical visualisation and biomechanical interpretation.

## Objectives

- Reproduce gait analysis calculations in Python
- Improve scientific programming skills
- Explore computational biomechanics
- Produce clear plots and results
- Develop reusable analysis workflows

## Features

- Gait cycle analysis
- Joint angle visualisation
- Numerical calculations
- Data processing
- Graph generation
- Biomechanical interpretation

## Skills Demonstrated
