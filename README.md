# Biomechanical Gait Analysis in Python

## Overview

This repository contains a Python-based reimplementation of a university biomechanics gait analysis project.

The original laboratory assignment was independently reproduced and extended to strengthen scientific programming skills while applying engineering principles to human movement analysis.

The project combines numerical methods, biomechanical analysis, data processing and scientific visualisation using Python.

---

## Objectives

- Apply Python to biomechanical data analysis.
- Reproduce gait analysis calculations.
- Develop reusable scientific programming workflows.
- Improve computational engineering skills.
- Strengthen understanding of human movement analysis.
